Apa itu turunan?

Turunan adalah konsep lanjutan dari limit. Turunan dapat disebut juga sebagai diferensial, sementara proses dalam menentukan turunan suatu fungsi disebut sebagai diferensiasi. Konsep turunan dapat dimaknai sebagai fungsi yang dapat disimbolkan sebagai “ƒ” (dibaca "ƒ aksen") yang nilainya pada sembarang bilangan c. ada beberapa macam rumus turunan lainnya, di antaranya sebagai berikut:

1. f(x) = k f'(x) = 0, ini turunan dari f(x) = c.
2. f(x)=x f'(x) = 1, ini turunan dari f(x) = x.
3. f(x) = kg(x) f'(x) = kg'(x), ini turunan dari f(x) = ax^n.
4. f(x)=x" f'(x) = nx"-1, ini turunan dari h(x) = f(x) + g(x).
5. f(x) = u(x) + v(x) f'(x) = u'(x) + v'(x), ini turunan dari h(x) = f(x) – g(x).
6. f(x) = u(x). v(x) f'(x) = u'(x) v(x) + u(x) v'(x), ini turunan dari f(x) = u(x) . v(x).
